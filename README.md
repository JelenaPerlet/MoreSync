# MoreSync 🌸

Mehr im Einklang mit deinem Körper.

MoreSync ist ein Begleiter für zyklusbasiertes Leben — Ernährung, Fasten, Training, Blutzucker, Tagebuch und Menopause-Unterstützung. Basierend auf den Büchern von Dr. Mindy Pelz (*Fast Like a Girl*, *Eat Like a Girl*, *Age Like a Girl*), Stacy Sims (*ROAR*) und Jessie Inchauspé (*Glucose Revolution*).

## Funktionen
- 🌙 Zyklus-Tracking mit Kalender und automatischer Phasenerkennung
- ⏰ Fasten-Timer mit allen Kuren und Erklärungen
- 🥗 Über 90 buchkonforme Rezepte, nach Phase gefiltert
- 💪 Phasenbasiertes Training (Zyklus & Menopause)
- 🩸 Blutzucker-Hacks
- 📔 Tagebuch mit Stimmung, Energie, Symptomen & Wasser-Tracker
- 🌸 Eigener Wechseljahre-Modus (Mondzyklus + Primal Menopausal Diet)

Alle Daten bleiben privat auf dem Gerät der Nutzerin.

## Technik
Gebaut mit React + Vite. Als PWA installierbar.

```bash
npm install
npm run dev      # lokal entwickeln
npm run build    # für Produktion bauen (Ordner: dist)
```
